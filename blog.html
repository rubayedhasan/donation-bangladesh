Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the page so that programs can manipulate its structure, style, and content. It converts the HTML document into a tree of objects that can be accessed and modified by JavaScript.

How do you select an element from the DOM?

You can select an element from the DOM using various methods, depending on how you want to target the element:

  1. document.getElementById(id): Selects an element by its unique id.
  2. document.querySelector(selector): Selects the first element that matches a CSS selector (like class, id, or element type).
  3. document.querySelectorAll(selector): Selects all elements that match the CSS selector and returns a NodeList.
  4. document.getElementsByClassName(className): Selects elements with a specific class name.
  5. document.getElementsByTagName(tagName): Selects all elements of a specified tag type (e.g., div, p).

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element, rather than attaching individual event listeners to each child element. This is possible because events bubble up through the DOM tree. For example, instead of adding a click listener to every individual button inside a list, you can add it to the parent

    and check which child element triggered the event using event.target. This is useful because it reduces memory consumption and improves performance, especially in cases where the DOM content changes dynamically (e.g., adding or removing list items), as you don't need to reattach event listeners to new elements.

How do you manipulate an element's attributes and styles using the DOM?

Manipulating Attributes: You can use methods like setAttribute() and getAttribute() to change or retrieve the values of an element's attributes. For example, element.setAttribute('src', 'new-image.jpg') changes the src attribute of an image element.
